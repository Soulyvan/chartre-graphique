### Référence Complète

| Couleur | Hex | Usage |
|---|---|---|
| 🔵 Bleu Nuit | `#0A1628` | Fond principal — body, header, footer, hero |
| 🔷 Bleu Ardoise | `#1E3A5F` | Structure — cards, sections alternées |
| 💠 Cyan Subtil | `#64B5F6` | **Accent & branding** — titres, liens, CTA, état actif |
| 🟢 Vert Tech | `#3FB950` | **Fonctionnel** — succès, validation, badges techniques |
| ⚪ Blanc Cassé | `#F8F9FA` | Texte principal — paragraphes, titres sur fond sombre |
| ⚙️ Gris Neutre | `#94A3B8` | Texte secondaire — descriptions, labels, metadata |
| ⚫ Noir Profond | `#0D1117` | Univers dev — blocs de code, style GitHub |

---

## 📐 Règles d'Utilisation des Couleurs

```
✅ Une seule couleur d'accent par composant
✅ Cyan  → identité, action, navigation
✅ Vert  → fonctionnel, technique uniquement
✅ Bleu nuit → base visuelle constante

❌ Ne jamais mettre cyan ET vert en concurrence directe
❌ Ne pas utiliser le vert pour décorer — réservé au code/succès
```

## 🔤 Typographie

| Rôle | Police | Poids | Usage |
|---|---|---|---|
| **Titres & Corps** | `Archivo` | 300 / 500 / 700 / 900 | Tout le contenu principal |
| **Code & Mono** | `Space Mono` | 400 / 700 | Snippets, valeurs hex, labels techniques |

```css
/* Import */
@import url('https://fonts.googleapis.com/css2?family=Space+Mono:wght@400;700&family=Archivo:wght@300;500;700;900&display=swap');

/* Usage */
font-family: 'Archivo', sans-serif;    /* Corps */
font-family: 'Space Mono', monospace; /* Code */
```

## ⚠️ À Ne Pas Faire

```
❌ Changer les couleurs (cyan #64B5F6 et vert #3FB950 sont intouchables)
❌ Déformer ou étirer le logo
❌ Ajouter des effets 3D ou ombres portées lourdes
❌ Utiliser sur des fonds trop chargés
❌ Espace autour du logo < 20px
❌ Inverser le sens des connexions ou des ondes
```
